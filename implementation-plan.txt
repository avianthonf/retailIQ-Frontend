### 3.1 `src/api/client.ts`
─────────────────────────────────────────

The Axios instance. Use the skeleton from Oracle Section 6.3 as the foundation
and apply all of the following:

**Concurrent-401 guard**: Use a module-level `isRedirecting` boolean flag.
Before calling `window.location.assign('/login')`, check
`if (isRedirecting) return; isRedirecting = true;`. Reset it never — a redirect
is terminal. This prevents multiple concurrent 401s from triggering multiple

**Upload-aware retry**: Do NOT retry file upload requests (requests whose
